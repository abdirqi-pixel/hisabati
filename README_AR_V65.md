# حساباتي v65

هذه نسخة Flutter مبنية على الملف الذي أرسلته: `hisabati_starter_fix3_code_cleanup.zip`.

## محتوى المشروع
- واجهة عربية RTL.
- GoRouter للتنقل.
- Riverpod لإدارة الحالة.
- SQLite عبر sqflite.
- مشاريع، أشخاص، تصنيفات، مصروفات، صندوق، تقارير.
- دعم PDF وExcel.
- نسخ احتياطي واستعادة.
- إشعارات وتذكيرات.
- PIN وقفل التطبيق وسجل أمان.
- شاشة بحث، مرفقات، إيرادات، سلف، ميزانيات، KPI، وتحليلات.

## إصلاحات v65
- تحديث رقم البناء إلى `1.0.0+65`.
- إزالة تكرار حزمة `printing` من `dev_dependencies`.
- جعل بدء التطبيق أكثر أمانًا إذا فشلت تهيئة الإشعارات.
- إضافة تقرير فحص ثابت للنسخة.

## التشغيل على لابتوب
بعد فك الضغط داخل مجلد المشروع:

```bash
flutter pub get
flutter analyze
flutter run
```

إذا لم يكن داخل الملف مجلدات Android/iOS كاملة، نفّذ من داخل المجلد:

```bash
flutter create .
flutter pub get
flutter run
```

## ملاحظة للتابلت أو الآيباد
من التابلت تقدر تحفظ الملف وتفتحه لاحقًا، لكن بناء APK يحتاج جهاز عليه Flutter SDK مثل لابتوب Windows / Mac / Linux أو بيئة تطوير سحابية.
