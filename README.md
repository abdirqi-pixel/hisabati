# حساباتي - Hisabati Starter

هذا أول إصدار تأسيسي لمشروع تطبيق حساباتي.

## يحتوي على
- Flutter
- واجهة عربية RTL
- شاشة بداية
- لوحة تحكم
- شاشة مشاريع
- شاشة إعدادات
- ثيم وألوان أساسية
- GoRouter
- Riverpod

## التشغيل
```bash
flutter pub get
flutter run
```

- countries
