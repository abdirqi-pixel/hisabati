# قائمة فحص أوامر Flutter

## 1. تحميل الحزم
```bash
flutter pub get
```

## 2. تحليل الكود
```bash
flutter analyze
```

## 3. تشغيل التطبيق
```bash
flutter run
```

## 4. بناء APK
```bash
flutter build apk --release
```

## 5. بناء AAB
```bash
flutter build appbundle --release
```

## تسجيل النتائج
اكتب نتيجة كل أمر في ملف FLUTTER_COMMAND_RESULTS_AR.md.
