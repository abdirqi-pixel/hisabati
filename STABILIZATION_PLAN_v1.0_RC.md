# خطة التثبيت النهائي v1.0 RC

## الهدف
تحويل نسخة Beta إلى نسخة Release Candidate قابلة للاختبار قبل النشر.

## فحوصات أساسية
- تشغيل التطبيق من البداية حتى لوحة التحكم.
- اختبار إنشاء مشروع وشخص وتصنيف.
- اختبار إضافة مصروف وإيراد.
- اختبار الصندوق.
- اختبار السلف والديون.
- اختبار التقارير PDF وExcel.
- اختبار النسخ الاحتياطي العادي والمشفر.
- اختبار المزامنة الاختيارية.
- اختبار القفل وPIN والبصمة.
- اختبار الوضع الليلي.
- اختبار البحث الذكي.
- اختبار التحليلات واللوحة التنفيذية.
- اختبار OCR للفواتير.
- اختبار العمليات الدورية.
- اختبار الصلاحيات بين المدير والمستخدم والمشاهد.

## فحوصات الأداء

- فحص سرعة البحث.
