# حالات اختبار حساباتي v1.0

## الحسابات الأساسية
1. افتح التطبيق لأول مرة.
2. أكمل الإعداد الأولي.
3. أنشئ مشروعًا جديدًا.
4. أضف شخصًا وتصنيفًا.
5. أضف مصروفًا بعدة مبالغ وتفاصيل.
6. أضف إيرادًا.
7. تحقق من تغير الرصيد.

## التقارير
1. افتح التقارير العادية.
2. افتح التقارير المتقدمة.
3. طبّق فلتر مشروع.
4. طبّق فلتر شخص.
5. صدّر PDF.
6. صدّر Excel.
7. جرّب الطباعة.

## الأمان
1. فعّل PIN.
2. أغلق التطبيق وافتحه.
3. جرّب PIN خاطئ.
4. جرّب PIN صحيح.
5. فعّل القفل التلقائي.
6. افحص سجل الأمان.

## النسخ الاحتياطي
1. صدّر نسخة عادية.
2. استعد النسخة.
3. صدّر نسخة مشفرة.
4. جرّب كلمة مرور خاطئة.
5. استعد بكلمة مرور صحيحة.

## الفواتير OCR
1. صوّر فاتورة واضحة.
2. راجع البيانات المستخرجة.
3. احفظها كمصروف.
4. تحقق من وجود الصورة كمرفق.
